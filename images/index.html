open the exhibit

A virtual space for showcasing digital art in an immersive way. This project emphasizes the atmosphere and aesthetic of an art installation using real time rendering techniques. Built in OpenGL, the exhibit invites users to explore and interact with the work in a fully navigable 3D environment.

Project Description

This project combines traditional gallery sensibilities with the possibilities of real time 3D graphics. The virtual exhibit showcases digital artwork in a stylized, immersive space where viewers can move freely, observe pieces from different angles, and engage with the environment. Using OpenGL, the gallery is rendered with Blinn-Phong lighting, textured surfaces, and post processing effects such as CRT style distortion and color bleed, creating a nostalgic, analog era feel.

Graphics Tech

  • OpenGL Graphics Pipeline
  • Blinn-Phong Lighting
  • Texture Mapping
  • OBJ Model Loader
  • Custom CRT Post Processing Shader
  • Full camera and movement control
